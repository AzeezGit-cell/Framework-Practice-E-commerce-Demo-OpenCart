# OpenCart Automation Project

This project is a Selenium-based automation framework for testing the OpenCart e-commerce application.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Jenkins (CI)

## Features
- Page Object Model (POM) design
- Data-driven testing
- Automated test execution
- Extent Reports for reporting
- CI integration using Jenkins

## How to Run
1. Clone the repository
2. Open in Eclipse or any IDE
3. Run using:
   mvn clean test

## CI Integration
This project is integrated with Jenkins to automatically trigger test execution on code changes.
